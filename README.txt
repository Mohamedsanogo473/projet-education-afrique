# Projet : Impact des dépenses publiques en éducation en Afrique

## Objectif
Analyser l'effet des investissements publics en éducation 
sur les taux de scolarisation dans 54 pays africains (2000-2022).

## Outils utilisés
- SQL (DB Browser for SQLite)
- Stata 15
- Power BI Desktop

## Sources de données
- Banque Mondiale
- UNDP
